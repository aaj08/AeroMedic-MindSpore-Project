## Project Overview
Aero-Medic is an innovation project for the **Huawei ICT Competition 2025-2026**. It uses AI to provide immediate medical help via drones.

## Huawei Technologies Used
* **MindSpore:** For developing AI applications.
* **CANN:** To ensure high-performance AI computing.
* **ModelArts:** For model training and management.
* **Orange Pi:** Used as the hardware platform for interactive display.

## Features
* Real-time injury detection using **MindSpore Lite**.
* Cloud-Edge collaboration between **Huawei Cloud** and the drone.
* Open-source contribution for the Regional and Global Finals.
